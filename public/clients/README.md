# Client Logos

Please add the following logo files to this directory:

1. **npdc-logo.png** - Nigerian Petroleum Development Company logo
2. **julius-berger-logo.png** - Julius Berger Nigeria PLC logo
3. **chevron-logo.png** - Chevron Corporation logo
4. **nddc-logo.png** - Niger Delta Development Commission logo
5. **lng-nigeria-logo.png** - LNG Nigeria Limited logo
6. **shell-logo.png** - Shell Petroleum Development Company logo

## Logo Guidelines

- **Format**: PNG with transparent background (preferred) or JPG
- **Size**: Recommended width: 200-300px, height will auto-scale
- **Quality**: High resolution for crisp display
- **Color**: Full color logos (grayscale effect applied on hover)

## Fallback Behavior

If a logo file is missing, the component will automatically display the client name as text instead.
